# Ruslan Helper Bot — локальный запуск на Windows

## Что тебе нужно один раз поставить

1. **Python 3.11 или новее** — https://www.python.org/downloads/
   При установке **обязательно** галочка «Add Python to PATH».

2. **Ollama** (для бесплатного локального ИИ Llama) — https://ollama.com/download
   После установки в PowerShell:
   ```
   ollama pull llama3.1
   ```
   Ollama сама стартует в трее. Проверка: открой http://localhost:11434 в браузере — должно ответить.

## Подготовка бота

1. Скачай папку `telegram-bot` (через Replit → Files → Download as zip → распакуй) в `C:\ruslan-bot\`.

2. Скопируй `.env.example` → `.env` и впиши свой `TELEGRAM_BOT_TOKEN`. Минимально достаточно его и `LLM_BACKEND=llama`.

3. Двойной клик по `run.bat`. При первом запуске он сам создаст виртуальное окружение и поставит зависимости (займёт минуту). После — просто стартует бота.

Окно закрыто = бот выключен. Чтобы бот работал всегда — добавь `run.bat` в автозагрузку Windows (`Win+R` → `shell:startup` → закинь туда ярлык).

## Что бот умеет на ПК

Управление компьютером доступно прямо в чате — пиши обычными словами:

- «открой ютуб», «зайди на gmail», «открой replit.com» → откроется в дефолтном браузере.
- «найди файл с налогами», «где у меня презентация про такси» → поиск по имени файла в `Desktop`, `Documents`, `Downloads`.
- «найди где у меня написано про Тоху» → поиск по содержимому документов (.txt, .md, .docx нужно конвертить, .pdf не сканирует — только текстовые).
- «открой папку C:\Работа» → открывает в проводнике.

Папки для поиска можно поменять в `.env`:
```
PC_SEARCH_DIRS=C:\Users\Ruslan\Desktop;D:\Работа
```

## Что НЕ работает локально

- **«📗 Таблицы», аналитика, монитор Google Sheets** — это требует отдельного API-сервера (Node.js), который сейчас крутится на Replit. Если нужно — скажи, перенесу.
- **Twilio SMS/звонки** — работают, если в `.env` прописаны `TWILIO_*` ключи.
- **Голосовой ответ (TTS)** — нужен `AI_INTEGRATIONS_OPENAI_*` или другой OpenAI-совместимый ключ. Распознавание голосовых сообщений требует OpenAI Whisper API.

## База данных

По умолчанию — SQLite, файл `data.db` рядом с ботом. Никаких настроек не нужно.
Чтобы использовать Postgres — пропиши в `.env`:
```
DATABASE_URL=postgresql://user:pass@host:5432/db
```

## Если что-то сломалось

- В окне `run.bat` видно все ошибки — скриншот пришли.
- `data.db` можно открыть программой `DB Browser for SQLite` — там лежат напоминания и долгосрочная память.
- Логи Ollama: `ollama serve` в отдельном окне покажет, что приходит к модели.
